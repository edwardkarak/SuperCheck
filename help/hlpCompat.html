  • Free Edition Compatability

SuperCheck is also compatible with the data.bin file, generated by SuperCheck Lite(Free). During program installation, the installer will find the data.bin file, convert it into another file, datanew.bin, and then delete the old file. The new file is now a valid SuperCheck Paid file, which can be opened and edited as though it was a regular file. Note that SuperCheck Paid files are not compatible with SuperCheck free edition.
