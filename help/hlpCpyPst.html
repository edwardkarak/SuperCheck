You can copy and paste transactions. This allows the same transaction to be duplicated, as many times as you need.

Cut

To cut a transaction, first seect the transaction, and then do one of the following:

  • Press CTRL + X
  • Press "Cut" in the "Edit" menu
  • Press "Cut" from the right-click menu

Now that you have cut the transaction, it should disappear from the table.

Copy

To copy a transaction, first select it, and then do one of the following:

  • Press CTRL + C
  • Right click the transaction and select "Copy" from the context menu that appears
  • Press "Copy" in the "Edit" menu

Paste

Now that there is a valid transaction in the clipboard, you can paste it, to complete the process. To paste a transaction:

  • Press CTRL + V
  • Right click the empty white space, and select "Paste" from the context menu that appears
  • Select "Paste" from the "Edit" menu

The pasted transaction will appear at the end of the transactions table.

Note that data that is not valid transaction data will not be pasted. For example the text "Foo" will not be pasted; nothing will happen.
