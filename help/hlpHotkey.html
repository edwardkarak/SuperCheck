Ctrl + Insert for Copy, Shift + Insert for Paste, and Shift + Delete for Cut also work.

Using the Arrow Keys to Navigate

You can use the arrow keys to navigate the table of transactions. Simply press the "Up" arrow key to move up one transaction, and press the "Down" arrow key to move down one transaction.
