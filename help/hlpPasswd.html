You may protect an important account with a password, so others may not be able to view it, unless they know the password.

To set up password protection, first open the file you want to protect. Next, go to "File->Password-Protect". When this is done, a dialog like this will be shown:

**insert screenshot**

In the first box, enter the desired password. It will be masked with black dots, or asterisks (the * character). In the second box, re-enter the password. When you are done, press the "OK" button. If the passwords in the two boxes match exactly, the password will be created. You must save for the password change to take effect.

Note: Passwords are case-sensitive

The next time you try to open the account, and it has a password, you will be presented with a dialog that asks for the password. The password will be masked with asterisks.

Changing/Clearing the Password

When you want to change the password, open the file to change, then go to "File->Password-Protect". Since the file is already password-protected, you will need to enter your current password. When you have done so, change the password boxes as needed. To clear the password, press the "Clear Password" button. You will need to save for changes to take place.
