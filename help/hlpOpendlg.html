When you press "Open", from the "File" menu, you will see a dialog like the one shown below:

screenshot
Number Explanation
1 This is a list of accounts you may edit. Double-click on the account you want, and it will be displayed in the main window.
2 This button will show a box asking for the new name you want to give the file. Enter the new name, and press "OK". The file will be renamed, and the list will update. Note that when no file is selected, this button will be disabled.
3 This will delete the selected file, and the file will be removed from the list. Note that when no file is selected, this button will be disabled.
4 This label shows the sum of the end balances of all the accounts in the list. So if there are 4 accouts whose end balances are $100, $80, $-200, and $556, then the value displayed here will be $536.
5 Opens the selected file, closes the Open dialog, and fills the main window with the file's contents. Note that when no file is selected, this button will be disabled.
