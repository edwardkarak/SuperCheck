Change Default Font... The default font of the table. You do not need to restart for font changes to take effect.
Autosave every... When this button is checked, then SuperCheck will automatically save any changes every n minutes , where n is the amount of minutes specified in the edit box. If this button is not checked, no autosaving will occur.
