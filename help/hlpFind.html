At one point, one will accumulate great amounts of transactions, and it will become increasingly difficult to find the transaction you need. For this exists the "Find" feature, which can find transaction(s) based on one of the following columns:

  • Description
  • Category

To begin searching, go to Edit->Find, and then select the column you want to search by. When you do so, a Find Dialog will appear, similar to this:

**screenshot**
Component Explanation
Find what: In this box, enter the search term. So if you need to find all transactions whose category contains "Foot", type "Foot" in this box.
Match case When this box is checked, then SuperCheck will perform a case-sensitive search. So "Foot" will bring up "Foot" and "Footwear", but not "foot", nor "footwear". (Notice the emphasis). By default, this box is checked.
Find Next When this button is pressed, the next transaction that matches the search criteria is selected, in gray. Keep pressing "Find Next" to find the next transaction. You will be notified if no more transactions match. This button is disabled (grayed out) until you type in something in the "Find what: " field.
Cancel Cancels the search operation. You can achieve this same effect by pressing the red "X" in the upper right corner.
