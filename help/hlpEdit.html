SuperCheck allows you to edit transactions. This is convenient if you have made a mistake. To edit a transaction,

  • Double click on the transaction
  • Right click the transaction, and then select "Edit Transaction" from the context menu that appears
  • Press the "Edit Transaction" icon in the toolbar (a pencil icon) while the transaction is selected
    • or
  • Press "Edit Transaction" in the "Edit" Menu

A dialog similar in appearance to the New Transaction dialog will appear, except all the fields will be filled in with the appropriate data. You can change these fields as needed. To learn how to fill out the fields, see Adding Transactions.

Once you have edited the transaction(s), it is recommended that you save your changes, either by pressing CTRL + S, the Save icon in the toolbar, or the "Save" in the "File" menu. (Unless you have made mistakes).
